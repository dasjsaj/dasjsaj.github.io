# Shengchao Zhu Homepage

This folder is a clean GitHub Pages + Jekyll personal academic homepage.

## How to Use

1. Create a GitHub repository named `your-github-username.github.io`.
2. Upload the files in this folder to the repository root.
3. Go to **Settings -> Pages**.
4. Set the source to **Deploy from a branch**.
5. Select the `main` branch and the `/root` folder.
6. Wait for GitHub Pages to build the site.

Do not upload the parent folder directly, because it contains private documents unrelated to the public homepage.

## Files

- `_config.yml`: Jekyll site configuration.
- `index.md`: Homepage content written in Markdown.
- `_layouts/default.html`: Page layout template.
- `assets/css/style.css`: Website styling.

## Optional Updates

- Add a profile photo by placing it under `assets/img/` and updating `_layouts/default.html`.
- Add GitHub and Google Scholar links after confirming the public profile URLs.
- Rename or expand sections in `index.md` as your academic profile grows.
